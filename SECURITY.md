# Security Policy

## Supported Versions

Currently, only the latest version of ClipSnap is supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| Latest  | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability within ClipSnap, please report it to us by creating a GitHub issue or contacting the maintainers directly.

We take all security reports seriously and will investigate and address them as quickly as possible. Please do not disclose vulnerabilities publicly until we have had a chance to fix them.
